# Automação de Relatórios de Vendas com Python

## 📌 Descrição

Este projeto consiste em uma automação desenvolvida em Python para gerar relatórios de vendas a partir de uma planilha Excel.

O script lê os dados de vendas, processa as informações e cria automaticamente uma nova planilha com métricas importantes para análise.

O objetivo é demonstrar como automatizar tarefas manuais e transformar dados brutos em informações úteis de forma rápida e eficiente.

---

## 🚀 Tecnologias utilizadas

* Python
* Pandas
* OpenPyXL

---

## ⚙️ Como funciona

1. O script lê uma planilha de vendas existente
2. Processa os dados utilizando Pandas
3. Calcula métricas importantes, como:

   * Faturamento total
   * Produto mais vendido
   * Vendedor com melhor desempenho
4. Gera automaticamente uma nova planilha Excel com os resultados

---

## 📄 Exemplo de saída

O script gera um arquivo Excel contendo:

* Resumo de vendas
* Indicadores principais
* Dados organizados para análise

---

## ▶️ Como executar o projeto

1. Instale as dependências:

```id="x1r9sq"
pip install pandas openpyxl
```

2. Execute o script:

```id="s4y7bn"

```

3. O relatório será gerado automaticamente na pasta do projeto.
